# **App Name**: Mindful Mosaic

## Core Features:

- Emotion Logging: Log daily emotions using a color picker and optional notes, storing data locally.
- Mosaic Art Generation: Automatically generate unique abstract mosaics based on logged color data and emotion trends using p5.js. and chroma.js.
- Anonymous Message Wall: Send and view anonymous supportive messages related to daily themes, moderated by emotion keywords.
- Daily Emotion Themes: Display daily themes to inspire reflection and shape the mosaic palette.
- Emotion Questionnaire: Present a quick questionnaire to fine-tune art mappings based on stress, self-care, and energy levels.
- Community Gallery: Anonymously share mosaics publicly with emotion tags to view others' creations filtered by color, emotion, or pattern type.
- Mood-Based Recommendations: Leverage the Spotify API tool to recommend playlists based on emotional trends. The tool decides when or whether this will improve user outcomes, so will use it's judgment to improve usability.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke calmness and introspection.
- Background color: Light gray (#E0E0E0) to provide a neutral and unobtrusive backdrop.
- Accent color: Soft lavender (#E6E6FA) to add a touch of serenity and visual interest.
- Body and headline font: 'Literata' for its literary, vintage, and slightly formal feel.
- Use minimalist and symbolic icons representing emotions and themes.
- Implement subtle animations for gradient transitions and mosaic pulsing, reflecting energy levels.
- Ensure a clean, intuitive layout with a focus on visual harmony and user-friendly navigation.
